🛠️ /spec & /task

O /spec planeja os buracos na estrada sem mover uma pedra. O /task é quem engata a primeira e obedece as regras desenhadas pelo /spec antes sem improvisos.

🧹 /close-sprint

O /close-sprint é invocado no final. Ele nunca toca em lógica comercial do app. Apenas varre os arquivos soltos, avisa a página WEB do Kanban que terminou, commita no GitHub e desliga as chaves.

💾 /compact55

O salva-vidas da sessão. Quando o Agente avisa que sua Token Window bateu em amarelo (>55%), dispare-o para extrair o estado bruto do sistema em Markdown, limpar o CLI e colar numa aba nova!
